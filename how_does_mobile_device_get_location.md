# 移动设备怎样定位
## 现象
### 没有GPS接收器的设备可以定位

- 云翼的iPad 1代 Wifi版（型号MD292CH），在接入公司wifi的情况下，可在地图软件上定位到“我的位置”
- 五福的三星手机，不开GPS模块，但接入了Wifi和移动运营商，可在百度地图上定位到“我的位置”

### 不接入网络可以定位

- 吴笑的iPhone 4S，关闭无线网和蜂窝数据（包括2G，3G），可以定位
- 吴笑的车载导航，只有GPS接收器，没有任何网络接入，可以定位导航
